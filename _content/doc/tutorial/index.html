Se você é uma pessoa nova em alguma parte do Go, dê uma olhada nos tutoriais abaixo.

Se você ainda não instalou o Go, consulte Baixe e instale.

Tutorial Descrição
Começando Diga Olá, mundo com Go.
Crie um módulo Um tutorial em várias partes que apresenta recursos comuns de linguagens de programação da perspectiva do Go.
Introdução aos espaços de trabalho multimódulos Apresenta os conceitos básicos de criação e utilização de espaços de trabalho multimódulos em Go. Espaços de trabalho multimódulos são úteis para fazer alterações em vários módulos.
Acessando um banco de dados relacional Apresenta os conceitos básicos de acesso a um banco de dados usando a biblioteca padrão.
Desenvolvendo uma API RESTful com Go e Gin Apresenta os conceitos básicos para escrever uma API de serviço web RESTful com Go e o framework web Gin.
Introdução a generics Apresenta os conceitos básicos de generics em Go. Com generics, você pode declarar e usar funções ou tipos que são escritos para funcionar com qualquer um de um conjunto de tipos fornecido pela código que faz a chamada.
Introdução ao fuzzing Apresenta os conceitos básicos de fuzzing em Go. O fuzzing pode gerar entradas para seus testes que podem identificar casos extremos e problemas de segurança que você pode ter perdido.
Introdução ao govulncheck Apresenta como encontrar e corrigir vulnerabilidades com o govulncheck. O govulncheck relata vulnerabilidades conhecidas que afetam o código Go.
Encontre e corrija dependências vulneráveis com o VS Code Go Apresenta como encontrar e corrigir dependências vulneráveis diretamente da sua IDE com o VS Code Go e o Vim.
Um tour pelo Go Uma introdução interativa ao Go: sintaxe básica e estruturas de dados; métodos e interfaces; e primitivas de concorrência do Go.
