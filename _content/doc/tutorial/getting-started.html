Neste tutorial, você terá uma breve introdução à programação em Go. Ao longo do caminho, você irá:

  • Instalar o Go (se ainda não o fez).
  • Escrever um código simples "Olá, mundo".
  • Usar o comando go para executar seu código.
  • Usar a ferramenta de descoberta de pacotes do Go para encontrar pacotes que você pode usar em seu próprio código.
  • Chamar funções de um módulo externo.
Observação: Para outros tutoriais, consulte Tutoriais.

Pré-requisitos

  • Alguma experiência em programação. O código aqui é bem simples, mas ajuda saber um pouco sobre funções.
  • Uma ferramenta para editar seu código. Qualquer editor de texto que você tiver funcionará bem. A maioria dos editores de texto tem bom suporte para Go. Os mais populares são VSCode (gratuito), GoLand (pago) e Vim (gratuito).
  • Um terminal de comando. Go funciona bem usando qualquer terminal no Linux e Mac, e no PowerShell ou cmd no Windows.

Instale o Go

Basta seguir os passos de Baixe e instale.

Escreva algum código

Comece com o Olá, mundo.

  1. Abra um prompt de comando e acesse seu diretório inicial com o comando cd.

    No Linux ou Mac:

    cd

    No Windows:

    cd %HOMEPATH%
  2. Crie um diretório hello para seu primeiro código-fonte em Go.

    Por exemplo, use os seguintes comandos:

        mkdir hello
    cd hello
  3. Habilite o rastreamento de dependências para seu código.

    Quando seu código importa pacotes contidos em outros módulos, você gerencia essas dependências por meio do próprio módulo do seu código. Esse módulo é definido por um arquivo go.mod que rastreia os módulos que fornecem esses pacotes. Esse arquivo go.mod permanece com seu código, inclusive no repositório de código-fonte.

    Para habilitar o rastreamento de dependências para o seu código criando um arquivo go.mod, execute o comando go mod init, fornecendo a ele o nome do módulo em que seu código estará. O nome é o caminho do módulo.

    No desenvolvimento real, o caminho do módulo normalmente será o local do repositório onde seu código-fonte será mantido. Por exemplo, o caminho do módulo pode ser github.com/meu-modulo. Se você planeja publicar seu módulo para outras pessoas usarem, o caminho do módulo deve ser um local de onde as ferramentas Go possam baixar seu módulo. Para mais informações sobre como nomear um módulo com um caminho de módulo, consulte Gerenciando dependências .

    Para os propósitos deste tutorial, use apenas exemplo/ola. 

        $ go mod init example/hello
    go: creating new go.mod: module example/hello

  4. No seu editor de texto, crie um arquivo ola.go para escrever seu código.

  5. Cole o seguinte código no seu arquivo ola.go e salve o arquivo.

        package main

    import "fmt"

    func main() {
        fmt.Println("Olá, mundo!")
    }

    Este é o seu código Go. Neste código, você:

    • Declara um pacote main (um pacote é uma forma de agrupar funções e é composto por todos os arquivos no mesmo diretório).
    • Importa o popular pacote fmt, que contém funções para formatação de texto, incluindo impressão no console. Este pacote é um dos pacotes da biblioteca padrão que você obteve ao instalar o Go.
    • Implementa uma função main para imprimir uma mensagem no console. Uma função main é executada por padrão quando você executa o pacote main.

  6. Execute seu código para ver a saudação.

        $ go run .
    Olá, mundo!

    O comando go run é um dos muitos comandos go que você usará para realizar tarefas com Go. Use o seguinte comando para obter uma lista dos outros: 

        $ go help

Chame código em um pacote externo

Quando precisar que seu código faça algo que pode ter sido implementado por outra pessoa, você pode procurar um pacote que tenha funções que você possa usar em seu código.

  1. Torne sua mensagem impressa um pouco mais interessante com uma função de um módulo externo.
    1. Visite pkg.go.dev e procure por um pacote "quote" .
    2. Nos resultados da pesquisa, localize e clique na v1 do pacote rsc.io/quote (ele deve estar listado com "Outras versões principais" de rsc.io/quote/v4).
    3. Na seção Documentation, em Index, observe a lista de funções que você pode chamar do seu código. Você usará a função Go.
    4. No topo desta página, observe que o pacote quote está incluído no módulo rsc.io/quote.

    Você pode usar o site pkg.go.dev para encontrar módulos publicados cujos pacotes têm funções que você pode usar em seu próprio código. Os pacotes são publicados em módulos — como rsc.io/quote — onde outras pessoas podem usá-los. Os módulos são aprimorados com novas versões ao longo do tempo, e você pode atualizar seu código para usar as versões aprimoradas.

  2. No seu código Go, importe o pacote rsc.io/quote e adicione uma chamada à sua função Go.

    Após adicionar as linhas destacadas, seu código deverá incluir o seguinte: 

        package main

    import "fmt"

    import "rsc.io/quote"

    func main() {
        fmt.Println(quote.Go())
    }
  3. Adicione novos requisitos e hashes para o módulo.

    Go adicionará o módulo quote como um requisito, bem como um arquivo go.sum para uso na autenticação do módulo. Para mais informações, consulte Autenticando módulos na Referência de Módulos do Go. 

        $ go mod tidy
    go: finding module for package rsc.io/quote
    go: found rsc.io/quote in rsc.io/quote v1.5.2
  4. Execute seu código para ver a mensagem gerada pela função que você está chamando. 
        $ go run .
    Don't communicate by sharing memory, share memory by communicating.

    Observe que seu código chama a função Go, exibindo uma mensagem inteligente sobre comunicação.

    Quando você executou go mod tidy, ele localizou e baixou o módulo rsc.io/quote que contém o pacote que você importou. Por padrão, ele baixou a versão mais recente -- v1.5.2.

Escreva mais código
