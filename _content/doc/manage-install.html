Este tópico descreve como instalar várias versões do Go na mesma máquina, bem como desinstalar o Go.

Para outros conteúdos sobre instalação, você pode ter interesse em:

  • Baixe e instale -- A maneira mais simples de instalar e começar a usar.
  • Instalando o Go a partir do código-fonte -- Como verificar o código-fonte, compilá-lo em sua própria máquina e executá-lo.

Instalando várias versões do Go

Você pode instalar várias versões do Go na mesma máquina. Por exemplo, você pode querer testar seu código em várias versões do Go. Para uma lista de versões que você pode instalar dessa forma, consulte a página de download.

Observação: para instalar usando o método descrito aqui, você precisará ter o git instalado.

Para instalar versões adicionais do Go, execute o comando go install, especificando o local de download da versão que deseja instalar. O exemplo a seguir ilustra a versão 1.10.7: 

$ go install golang.org/dl/go1.10.7@latest
$ go1.10.7 download

Para executar comandos go com a versão recém-baixada, anexe o número da versão ao comando go, da seguinte maneira: 

$ go1.10.7 version
go version go1.10.7 linux/amd64

Quando você tem várias versões instaladas, é possível descobrir onde cada uma está instalada observando o valor GOROOT da versão. Por exemplo, execute um comando como o seguinte: 

$ go1.10.7 env GOROOT

Para desinstalar uma versão baixada, basta remover o diretório especificado pela variável de ambiente GOROOT e o binário goX.Y.Z.

Desinstalando o Go

Você pode remover o Go do seu sistema usando as etapas descritas neste tópico.

Removendo configuração e dados da pessoa usuária

O Go armazena a configuração da pessoa usuária no diretório go dentro do diretório de configuração da pessoa usuária, conforme retornado por os.UserConfigDir. Este também pode ser encontrado como o diretório que contém o arquivo de configuração retornado por go env GOENV.

O Go armazena artefatos de compilação intermediários no diretório retornado por go env GOCACHE. Eles podem ser removidos com go clean -cache.

O Go armazena as dependências baixadas no diretório retornado por go env GOMODCACHE. Elas podem ser removidas com go clean -modcache.

Linux / macOS / FreeBSD

  1. Exclua o diretório go.

    Geralmente é /usr/local/go.

  2. Remova o diretório bin do Go da sua variável de ambiente PATH.

    No Linux e FreeBSD, edite /etc/profile ou $HOME/.profile. Se você instalou o Go com o pacote do macOS, remova o arquivo /etc/paths.d/go.

Windows

A maneira mais simples de remover o Go é por meio de Adicionar/Remover Programas no painel de controle do Windows:

  1. No Painel de Controle, clique duas vezes em Adicionar/Remover Programas.
  2. Em Adicionar/Remover Programas, selecione Linguagem de Programação Go, clique em Desinstalar e siga as instruções.

Para remover o Go com ferramentas, você também pode usar a linha de comando:

  • Desinstale usando a linha de comando executando o seguinte comando: 
    msiexec /x go{{version}}.windows-{{cpu-arch}}.msi /q

    Observação: Usar este processo de desinstalação para Windows removerá automaticamente as variáveis de ambiente do Windows criadas pela instalação original.
